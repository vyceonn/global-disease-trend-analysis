# Global Disease Trend Analysis

Analysis of 429,000+ rows of real COVID-19 epidemiological data using Python.

## Tools Used
- Python, Pandas, Matplotlib, Seaborn

## What This Project Does
- Fetches and cleans real COVID-19 data from Our World in Data
- Handles 429,435 rows across 211 countries (2020–2024)
- Generates 4 visualizations:
  - Case trend lines for top 5 countries
  - Deaths per million bar chart
  - Vaccination rate vs mortality scatter plot
  - Seasonal heatmap of new cases by month/year
- Calculates correlation between vaccination rates and mortality across 211 countries

## Key Finding
Correlation analysis across 211 countries revealed that the relationship between vaccination rates and mortality is influenced by confounding factors such as GDP and reporting quality — demonstrating the complexity of real-world epidemiological data.
